# ByteCat Solutions - Proyecto TFG Ciberseguridad

Este repositorio contiene el código fuente, scripts y configuraciones utilizadas en el proyecto de ciberseguridad.

## Contenido

- web-app: aplicación web vulnerable
- scripts: herramientas utilizadas
- configs: configuraciones de Nginx y Wazuh

## Entorno del laboratorio

- SRV-WEB-DMZ
- SRV-SIEM-WAZUH
- Kali Linux
- Nginx
- Flask
- SQLite
- Wazuh SIEM

## Objetivo

Simular un entorno corporativo vulnerable, ejecutar ataques (Red Team) y detectar la actividad mediante herramientas de monitorización (Blue Team).

## Nota

Proyecto desarrollado en entorno controlado con fines educativos.
